# MCP settings

## Include MCP settings on query level

```yaml
apiVersion: ark.mckinsey.com/v1alpha1
kind: Query
metadata:
  name: my-query
  annotations:
    # Immediately call set_base_directory tool with path "shared" after initiating connection to filesystem
    "ark.mckinsey.com/mcp-server-settings": |
      {"ark-system/filesystem": {"toolCalls": [{"name": "set_base_directory", "arguments": {"path": "./shared"}}]}}
spec:
  input: "Hello"
  targets:
    - name: my-agent
```
